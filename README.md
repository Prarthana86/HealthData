📊 Data Visualization Dashboard for File Usage & Engagement Analysis

This project presents an interactive data visualization dashboard designed to analyze file usage patterns, author contributions, and engagement metrics such as upvotes and usability ratings. The dashboard provides meaningful insights through visually appealing charts and graphs, enabling better understanding of data trends and user behavior.

🚀 Key Features
Author-wise Analysis: Visual breakdown of total upvotes by different authors
Engagement Metrics: Displays overall upvotes and usability ratings in summary cards
Yearly Trends: Line chart showing how upvotes evolve over time
File Type Distribution: Pie chart illustrating the proportion of different file formats (CSV, JSON, etc.)
Interactive Filters: Filter data by author, upload date, and file type
Comparative Insights: Combined visualization of upvotes vs usability ratings

🛠️ Technologies Used
Data Visualization Tool (e.g., Power BI / Tableau / similar)
Data Processing (Excel / CSV datasets)
Dashboard Design & Analytics

📈 Insights Provided
Identify top-performing authors based on upvotes
Understand which file types are most commonly used
Track growth or decline in engagement over the years
Compare usability ratings with popularity metrics

💡 Use Cases
Data analytics and reporting
Performance tracking of contributors
Understanding user engagement patterns
Academic or business intelligence projects
